# stringGrouper
normally frontend and backend should be in sperate folders, for the sake of simplicty I'm putting them in the same project for now

clone the project
## backend
* prerequisite: 
1. Python 3.7
2. pipenv 2020.6.2
* steps:
1. pipenv install
2. cd api
3. python3 app.py

## frontend
* prerequisite: 
1. node 12.16
2. npm 6.13
* steps:
1. cd vue_frontend
2. npm install
3. npm run serve
4. open http://localhost:8081/ in browser
